## Design Tools

Recommended tools for creating these PDFs:
- **Canva** (easiest, has templates)
- **Adobe InDesign** (professional)
- **Figma + Export to PDF** (design-first approach)
- **Google Docs** (simple, free)

## Checklist Before Publishing

- [ ] File names match exactly (hyphens, no spaces, lowercase)
- [ ] PDF file size under 2 MB
- [ ] All text is readable and not pixelated
- [ ] Nexprove logo included on all pages
- [ ] Contact information correct (info@nexprove.com, nexprove.com)
- [ ] No Lorem Ipsum or placeholder text
- [ ] Tested opening in multiple PDF readers
- [ ] Copyright notice included (© 2026 Nexprove)
